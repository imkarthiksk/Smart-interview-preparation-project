🤖 AI Resume Analyzer & Interview Preparation Platform
📌 Overview

AI Resume Analyzer & Interview Preparation Platform is a full-stack MERN application that helps candidates prepare for technical interviews.

The platform analyzes a user's resume, identifies missing skills, generates company-specific interview questions using AI, evaluates answers, and provides detailed interview reports.
